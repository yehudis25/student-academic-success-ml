# Student Dropout and Academic Success Prediction

This repository contains my final project for Machine Learning in Python.

The project compares the performance of five machine learning algorithms on the Predict Students' Dropout and Academic Success dataset from the UCI Machine Learning Repository.

## Models Evaluated

- Logistic Regression
- Decision Tree
- AutoML (PyCaret)
- Random Forest
- Neural Network (PyTorch)

## Repository Contents

- `student_success_prediction.ipynb` – Jupyter Notebook containing all code used for data exploration, model training, evaluation, and visualizations.
- `Yehudis_Schnaidman_Student_Dropout_ML_Final_Report.pdf` – Final project report summarizing the methodology, results, and conclusions.

## Dataset

UCI Machine Learning Repository:  
https://archive.ics.uci.edu/dataset/697/predict+students+dropout+and+academic+success
